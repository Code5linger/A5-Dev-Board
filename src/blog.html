Question-1: What are the different ways to select an element in the DOM?

  • getElementById()
  • getElementsByClassName()
  • getElementsByTagName()
  • querySelector()
  • querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: Gets or sets HTML content inside an element. innerText: Gets or sets visible text, excluding hidden elements. textContent: Gets or sets all text inside an element, including hidden text.

Question-3: What is event delegation in the DOM?

It's a technique where you attach a single event listener to a parent element instead of individual child elements. The event is then triggered when it bubbles up from the child.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event starts at the most specific element (child) and bubbles up to the root (parent) elements.

Question-5: How do you create, add, and remove elements using JavaScript?

Create: document.createElement('tag') Add: parent.appendChild(newElement) Remove: parent.removeChild(child)
